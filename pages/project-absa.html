Aspect-Based

Model

NLP Transformer Multi-Task Learning SemEval-2014

Developed K-Tran — a syntax-aware transformer extending RoBERTa with spaCy-based dependency attention. Implements joint Aspect Term Extraction and Sentiment Classification with a Gradio interface for real-time evaluation.

Architecture

System Flowchart

Fig 1. End-to-end pipeline of the Syntax-Aware K-Tran model · ATE = Aspect Term Extraction · ASC = Aspect Sentiment Classification
Objective

Develop an advanced ABSA system identifying aspect terms and their corresponding sentiments with high precision by integrating syntactic dependency information into transformer attention mechanisms.

Tech Stack
RoBERTa-basespaCy PythonGradio SemEval-2014PyTorch
Key Features
  • Syntax-aware attention bias matrix from spaCy parse trees
  • Joint multi-task learning for ATE and ASC simultaneously
  • BIO tagging scheme for aspect boundary detection
  • Weighted loss: L = α·L_ATE + (1−α)·L_ASC
  • Real-time evaluation via Gradio interface
Datasets
  • absa_30000.csv — ~30K annotated social media sentences
  • train.xml — ~3,000 SemEval-2014 restaurant reviews
  • test.xml — ~800 SemEval-2014 test sentences
  • Labels: Positive / Negative / Neutral polarity
Outcome

Demonstrated improved multi-aspect sentiment understanding and more accurate sentiment classification by effectively leveraging syntactic and contextual relationships within text. Evaluated on Accuracy, Precision, Recall, Macro-F1, Weighted F1 and ATE F1-score.
